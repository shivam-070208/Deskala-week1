## How to Run Code Locally

1. **Clone the repository** if you haven't already:

   ```bash
   git clone https://github.com/shivam-070208/Deskala-week1
   cd dsa-homework
   ```

2. **Install Node.js** if you don't have it:
   - [Node.js download page](https://nodejs.org/)

3. **Navigate to the directory of the question:**

   ```bash
   cd level14/easy
   # or
   cd level14/medium
   ```

4. **Run the TypeScript file directly with Node.js:**

   Node.js (version 20.6.0 and above) can run TypeScript files directly.
   Just use:

   ```bash
   node climbing-stairs.ts
   # or
   node house-robber.ts
   ```

   Alternatively, if you are using an older version of Node.js,
   you can use `ts-node` for direct execution:

   ```bash
   npm install -g ts-node
   ts-node climbing-stairs.ts
   # or
   ts-node house-robber.ts
   ```

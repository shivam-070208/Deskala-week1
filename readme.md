## How to Clone This Repository

To get started, you need to clone this repository to your local machine. Use the following commands:

```bash
git clone https://github.com/shivam-070208/Deskala-week1.git
cd Deskala-week1/dsa-homework
```

Now you can follow the instructions in the main `readme.md` to run individual files.
